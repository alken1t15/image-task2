# Исследование RGB-параллелизации

В этой папке я оставил только файлы, которые нужны для отчёта:

- `grid_benchmark.csv` — полный прогон на четырёх изображениях, четырёх фильтрах и потоках `1/2/4/8` для стратегии `grid`;
- `strategy_benchmark.csv` — сравнение стратегий `pixels`, `rows`, `columns`, `grid` на `2048.jpg` и `gaussian5`;
- `grid_time.svg` — график среднего времени для всех изображений и фильтров;
- `grid_speedup_2048.svg` — график ускорения на `2048.jpg`;
- `strategy_gaussian5_2048.svg` — график сравнения стратегий.

Исходные изображения использовались локально из `D:\temp`: `256.png`, `512.jpg`, `1024.jpg`, `2048.jpg`. В репозиторий я их не добавлял, потому что для проверки исследования достаточно CSV и графиков.
